# 🌐 Ionic Server Management

Use this workflow to start or stop the application during development.

---

## 🚀 Start Server

To launch the application in development mode with live reloading:

// turbo
1. Run the start command:
```powershell
cd c:\Users\HP\MKPrep\IonicProjects\utility-app
npm run start
```

2. Access the app:
Open your browser at [http://localhost:4200](http://localhost:4200).

---

## 🛑 Stop Server

To stop the running server:

1. Locate the terminal running `npm run start`.
2. Press `Ctrl + C` in the terminal.
3. Confirm termination if prompted (type `Y` and Enter).

---

## 🔁 Restart Server

If the server hangs or needs a fresh build:

1. Stop the server using the steps above.
2. Run the start command again.
