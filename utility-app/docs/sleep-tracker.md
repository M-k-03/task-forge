# Feature: Sleep Tracker (Nightly Rest Forge)

## 📌 Overview
The Sleep Tracker is a component of the Task Forge ecosystem designed to optimize recovery. It allows users to track their nightly sleep cycles, calculate efficiency, and maintain a consistent sleep rhythm.

---

## 🏗️ Requirements

### 1. Daily Tracking
- **Inputs:** Lights Out Time, Wake Up Time.
- **Calculations:** Total Sleep Duration, Sleep Efficiency (duration vs. ideal), Consistency (offset from average).

### 2. Dashboard Visualization
- **Hero Metric:** Last night's total hours.
- **Trend View:** 7-day bar chart showing sleep quantity.
- **Qualitative Assessment:** Option to rate sleep quality (e.g., scale 1-5).

### 3. Insights
- Average sleep duration for the week.
- Average "Lights Out" and "Wake Up" times.
- Motivational badge derived from consistency (e.g., "The Steady Rested").

---

## 🎨 Design Reference
- Use **Velocity Slates** for the background.
- Use **Deep Velocity Blue** for active trackers.
- Incorporate a soft "tonal blur" for the sleep cards.
- Rounded corners: **32px** (Round Thirty-Two) for cards.
